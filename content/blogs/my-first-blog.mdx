## 🚀 The Code (Taken from Aceternity)

Here’s how you can build it step by step:


```tsx
import { cn } from "@/lib/utils"

interface CardProps {
  className?: string
  children: React.ReactNode
}

export function Card({ className, children }: CardProps) {
  return (
    <div
      className={cn(
        "rounded-2xl border bg-background p-6 shadow-sm transition-all hover:shadow-md",
        className
      )}
    >
      {children}
    </div>
  )
}
```
